Image Classification using Haar Cascade, SVM, and Wavelet Transform
Overview

This project focuses on image classification by combining traditional computer vision techniques with machine learning. It uses Haar Cascade for feature extraction, Wavelet Transform for enhanced feature representation, and a Support Vector Machine (SVM) for accurate classification.

Features
Image preprocessing and feature extraction using Haar Cascade
Feature enhancement using Wavelet Transform
Classification using Support Vector Machine (SVM)
Efficient and lightweight approach compared to deep learning models
Suitable for real-time and resource-constrained applications
Technologies Used
Python
OpenCV (Haar Cascade)
Scikit-learn (SVM)
NumPy
PyWavelets (Wavelet Transform)
